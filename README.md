# 📱 Smartphone Data Analysis using SQL

## 📌 Project Overview

This project demonstrates end-to-end data analysis using SQL on a smartphone dataset. The objective was to clean the data, perform exploratory data analysis (EDA), identify business insights, and build reusable SQL objects to support data-driven decision-making.

## 🚀 Project Highlights

- Cleaned and prepared the smartphone dataset by identifying and removing duplicate records using **Common Table Expressions (CTEs)**.
- Performed **Exploratory Data Analysis (EDA)** to uncover trends, patterns, and anomalies within the smartphone data.
- Developed complex SQL queries using **Joins**, **Aggregate Functions**, and **System Functions** to extract meaningful business insights.
- Created reusable **Views** to simplify data retrieval and improve query efficiency.
- Designed **Stored Procedures** to automate repetitive analysis tasks and streamline reporting.
- Analyzed smartphone attributes such as **Brand, Price, RAM, Storage, Battery Capacity, Camera Specifications, and Ratings**.
- Applied SQL filtering, sorting, grouping, and conditional logic to answer business-related questions from the dataset.
- Optimized query readability and maintainability using SQL best practices and modular query design.
- Strengthened hands-on experience with relational database concepts including data cleaning, transformation, and analytical querying.
- Demonstrated practical SQL skills commonly used in Data Analyst and Business Intelligence roles.

## 🎯 Project Outcome

This project showcases practical SQL skills including data cleaning, exploratory analysis, query optimization, and database object creation. It demonstrates the ability to transform raw data into actionable insights using structured SQL techniques, making it suitable for Data Analyst, Business Analyst, and Business Intelligence portfolios.

  # 🛒 Sales of Supermarket Analysis using Python

---

## 🚀 Project Highlights

- Imported and processed supermarket sales data using Python for comprehensive business analysis.
- Cleaned and transformed raw data by handling missing values, duplicates, and inconsistent records to improve data quality.
- Performed Exploratory Data Analysis (EDA) to identify sales trends, customer behavior, and product performance.
- Created aggregated reports to evaluate revenue, sales distribution, and support investment and business decisions.
- Analyzed key performance indicators (KPIs) including Total Sales, Gross Income, Customer Ratings, Quantity Sold, and Product Line Performance.
- Developed insightful visualizations to identify patterns, seasonal trends, and opportunities for business growth.
- Utilized statistical analysis to compare sales across branches, customer types, payment methods, and product categories.
- Generated actionable business insights that can help improve sales performance and operational efficiency.
- Applied Python data analysis best practices to build a reproducible and well-structured analytical workflow.
- Demonstrated practical data analytics skills commonly used in Data Analyst, Business Intelligence, and Data Science roles.
